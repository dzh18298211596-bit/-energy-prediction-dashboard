# 单车能耗预测校准模型

基于机器学习的南区二车间60JPH能耗智能预测系统

## 在线访问

启用GitHub Pages后，访问：
https://dzh18298211596-bit.github.io/-energy-prediction-dashboard/energy_prediction_dashboard.html

## 本地使用

直接在浏览器中打开 `energy_prediction_dashboard.html` 文件即可使用。

## 功能特点

- 能源用量预测（电、气、水）
- 能源成本分析
- 影响因素重要性分析
- 历史数据对比分析
- 智能洞察和建议

## 启用GitHub Pages

1. 进入仓库设置（Settings）
2. 在左侧菜单找到 "Pages"
3. 在 "Build and deployment" 下：
   - Source: 选择 "Deploy from a branch"
   - Branch: 选择 "main" 分支，文件夹选择 "/ (root)"
4. 点击 Save
5. 等待几分钟后，页面将在 https://dzh18298211596-bit.github.io/-energy-prediction-dashboard/ 上线
